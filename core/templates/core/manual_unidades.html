Unidades

Gerencie os padrões de unidades de medida dos produtos do sistema.

Apenas para usuários Administradores!

01

Listagem

A tela principal de unidades permite visualizar o que está cadastrado no sistema.

02

Cadastrar Nova Unidade

Utilize essa função para cadastrar uma nova unidade.

Clique em Nova Unidade, preencha o formulário e clique em salvar.

Ao cadastrar uma nova unidade, ela ficará disponível para seleção na tela de cadastro/alteração de produtos.

03

Editar uma Unidade

Essa função é realizada através do Painel Administrativo.

Para acessá-lo, adicione "/admin" ao final do endereço do sistema no seu navegador.

Selecione Unidades e clique sobre a unidade desejada para realizar alterações.

Atenção: Alterar ou excluir unidades pode afetar o histórico de produtos vinculados. Use com cautela.
