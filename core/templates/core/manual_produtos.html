Produtos

Busca, cadastro e alteração de produtos no catálogo de materiais do escritório.

01

Listagem e Busca

A tela principal de produtos permite visualizar o que está cadastrado no sistema.

  • Filtro Automático: Por padrão, o sistema exibe os 5 itens com maior estoque.
  • Busca: Digite o nome do produto no campo de pesquisa para filtrar itens específicos.
02

Cadastro de Produtos

Utilize essa função para registrar novos itens no sistema.

Dica: Antes de cadastrar, verifique se o item já não está cadastrado para evitar duplicidade.

Clique em Novo Produto, preencha o formulário e clique em Salvar.

03

Alteração e Detalhes

Para editar um produto, visualize os detalhes clicando sobre o nome dele na lista e, em seguida, clique em Alterar Produto.
