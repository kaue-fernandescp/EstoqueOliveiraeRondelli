Usuários

Gerencie os usuários e níveis de acesso.

Apenas para usuários Administradores!

01

Listagem

A tela principal de usuários retorna todos os usuários cadastrados no sistema.

Essa tela mostra se o usuário está ativo ou inativo e se é administrador ou não.

02

Registro de Novos Colaboradores

Clique no botão Cadastrar Usuário para adicionar um novo usuário.

Defina um usernameúnico. O sistema registrará automaticamente quem criou o usuário para fins de auditoria.

03

Níveis de Permissão (Admin)

Ao marcar um usuário como Administrador, ele passará a ter acesso aos seguintes módulos:

  • Cadastro/alteração de unidades;
  • Gerenciamento de usuários;
  • Painel administrativo.
